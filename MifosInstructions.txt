With gradle project, tried to get it up and running with the provided steps.
Here's what needs to be done (run the below commands in command prompt):
1. git clone https://github.com/openMF/mifosx  (git clone https://github.com/reehanlalkhan/imifosx-server dev.git)
    After successful execution of the above command a folder named mifosx
 2. cd to mifosx/fineract-provider
 3. Run the below commands in the given order:
     a. gradlew -Penv=dev clean cleanEclipse eclipse
     b. gradlew -Penv=dev clean war
     (Note: With the above commands gradle will get installed and dependency jars will also be downloaded and hence will need loads of internet)
 4. Open eclipse and then do: import->General->Existing projects
      Select the folder of mifosx
 5. Run As -> Java Application this class: ServerWithMariaDB4jApplication
